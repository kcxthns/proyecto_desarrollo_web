Reglas iptables se guardan en /etc/sysconfig/iptables

Permitir conexiones hacia el puerto 80 de la maquina sobre protocolo TCP
iptables -A INPUT -p tcp --dport 80 -j ACCEPT
---------------------------------------------------------------------------
Bloquear telnet INPUT (port 22)
iptables -A INPUT -p tcp -m tcp --dport 23 -j DROP
---------------------------------------------------------------------------
Bloquear todas las conexiones
iptables -P INPUT DROP
iptables -P FORWARD DROP
iptables -P OUTPUT DROP
---------------------------------------------------------------------------
Ver reglas iptables
iptables -L -v -n
